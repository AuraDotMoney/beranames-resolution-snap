# beranames-resolution-snap
This repo includes the npm package to install as snap on metamask to resolve BNS - Berachain Name Service

## For run correctly

Open Beranames-Resolution-Snap Repository:
```bash
yarn install
```
then
```bash
yarn run build 
```
then 
```bash
yarn run serve
```

Then
Open Beranames-Snap Repository 
```bash
yarn install
```
then
```bash
yarn start
```
